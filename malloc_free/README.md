# malloc_free

This project covers dynamic memory allocation in C using `malloc` and `free`.

## Files

- `0-create_array.c` - Creates an array of chars and initializes it
- `1-strdup.c` - Returns a duplicate of a string
- `2-str_concat.c` - Concatenates two strings
- `3-alloc_grid.c` - Creates a 2D array of integers
- `4-free_grid.c` - Frees a 2D array

## Concepts

- Dynamic memory allocation
- malloc
- free
- NULL pointers
- Arrays and strings
- 2D arrays
