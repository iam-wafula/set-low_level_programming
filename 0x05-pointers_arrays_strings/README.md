# 0x05. C - Pointers, Arrays and Strings

This project covers the fundamentals of pointers, arrays, and strings in C programming.

## Learning Objectives

- What are pointers and how to use them
- Arrays and how they work
- Difference between pointers and arrays
- How to manipulate strings
- Memory addresses and dereferencing

## Tasks

0. reset_to_98 - Update value using pointer  
1. swap_int - Swap two integers  
2. _strlen - Return length of string  
3. _puts - Print string  
4. print_rev - Print string in reverse  
5. rev_string - Reverse a string  
6. puts2 - Print every other character  
7. puts_half - Print half of a string  
8. print_array - Print elements of an array  
9. _strcpy - Copy a string
