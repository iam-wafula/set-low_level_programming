# Low Level Programming - Singly Linked Lists

## Project Description
This project focuses on implementing singly linked lists in C as part of ALX Software Engineering program. It covers creation, traversal, insertion, deletion, and memory management of linked lists.

---

## Files in this project

### 0-print_list.c
Prints all elements of a linked list.

### 1-list_len.c
Returns the number of elements in a linked list.

### 2-add_node.c
Adds a new node at the beginning of a linked list.

### 3-add_node_end.c
Adds a new node at the end of a linked list.

### 4-free_list.c
Frees all memory allocated for the linked list.

### 100-first.c
Prints a message before main function is executed using constructor attribute.

### 101-hello_alx.asm
64-bit assembly program that prints "Hello, Frontier" using printf.
