# C - Debugging

## Description
This project focuses on debugging C programs by identifying and fixing errors in existing code.

## Learning Objectives
- Understand debugging in C
- Identify logical and syntax errors
- Test and fix programs

## Requirements
- Allowed editors: vi, vim, emacs
- Compiled using: gcc -Wall -Werror -Wextra -pedantic -std=gnu89
- Ubuntu 20.04 LTS
- All files must end with a new line
- Follow Betty coding style

## Tasks

0. Multiple Mains  
Created a test file to check that positive_or_negative() prints correctly when the input is 0.

1. Like, Comment, Subscribe  
Fixed an infinite loop by commenting out problematic code.

2. 0 > 972?  
Corrected the program to return the largest of three numbers.

3. Leap Year  
Fixed leap year handling in print_remaining_days().
