# More Singly Linked Lists

This project contains implementations of singly linked list operations in C as part of ALX Data Structures and Algorithms.

## Files

- 0-print_listint.c: Prints all elements of a listint_t list
- 1-listint_len.c: Returns number of elements in a list
- 2-add_nodeint.c: Adds node at beginning
- 3-add_nodeint_end.c: Adds node at end
- 4-free_listint.c: Frees a list
- 5-free_listint2.c: Frees list and sets head to NULL
- 6-pop_listint.c: Deletes head node and returns its data
- 7-get_nodeint.c: Returns node at a given index

gcc -Wall -pedantic -Werror -Wextra -std=gnu89 *.c -o output
